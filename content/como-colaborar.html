¿Representas a una delegación o quieres que tu delegación participe? Información para Delegaciones

¿Quieres colaborar en el Proyecto Alejandría? Existen distintos grupos de trabajo en los cuales puedes echar una mano. La participación es completamente voluntaria, aunque sí se pide consistencia y compromiso para poder mantener el proyecto estable.

Comisión inter-delegaciones

Este grupo de trabajo se compone de los representantes de las distintas delegaciones participantes. Su objetivo principal es el de coordinar los demás grupos de trabajo, e informar a las delegaciones y sus estudiantes del estado del proyecto.

Cómo participar

Cada delegación ha designado uno o más responsables para este grupo de trabajo, contacta con ellos a través de tu delegación para poder participar en este grupo.

Desarrollo de la Aplicación Web

Este grupo de trabajo se encarga de desarrollar la aplicación web como tal, por tanto, está compuesto de personas con conocimientos técnicos de desarrollo web e ingeniería de software.

Cómo participar

El desarrollo se lleva a cabo en la organización de GitHub del proyecto, la forma más rápida de participar es revisar qué tareas están pendientes en el repositorio en el que se desee participar y ponerse en contacto con los mantenedores de dicho repositorio para coordinar el trabajo.

Desarrollo del Modelo de Gestión de Contenido

Este grupo de trabajo diseña y crea el modelo de gestión de contenido para las delegaciones, es un trabajo que no requiere cualificación técnica, siendo, por tanto, el más accesible para nuevos voluntarios.

Cómo participar

El desarrollo se lleva a cabo en reuniones telemáticas o presenciales coordinadas por la organización del Proyecto, para participar, ponte en contacto con el responsable del Proyecto en tu delegación.
